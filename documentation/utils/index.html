    Modulo Utils

    Il modulo Utils raccoglie una serie di funzioni di utilità per la gestione e la conversione di file, la creazione di directory e l'estrazione di contenuti da archivi compressi. Queste funzioni sono fondamentali per supportare i processi di gestione dei dati censuari e geografici.

    census_folder(output_data_folder, year)

    Crea (se necessario) la cartella dedicata ai dati del censimento per un anno specifico.

    La funzione genera una directory denominata census_<anno> all’interno della cartella output_data_folder e la crea se non esiste già. Questa cartella rappresenta la radice dei dati scaricati ed elaborati per un determinato censimento, mantenendo una struttura ordinata e coerente tra anni diversi.

    PARAMETER DESCRIPTION
    output_data_folder

    Cartella principale sotto cui creare la directory del censimento.

    TYPE: Path

    year

    Anno del censimento da organizzare (es. 1991, 2001, 2011, 2021).

    TYPE: int

    RETURNS DESCRIPTION
    Path

    Il percorso completo della cartella del censimento creata o già esistente.

    TYPE: Path

    RAISES DESCRIPTION
    Exception

    Se la cartella non può essere creata per motivi di permessi o percorsi non validi.

    check_encoding(data)

    Determina la codifica di un file leggendo un campione iniziale.

    La funzione apre il file in modalità binaria, legge i primi 100.000 byte e utilizza la libreria chardet per stimare la codifica del testo. Nel caso in cui chardet identifichi la codifica come 'ascii', questa viene convertita in 'latin1' per garantire maggiore compatibilità, poiché molti file amministrativi e geografici possono contenere caratteri estesi pur essendo formalmente interpretati come ASCII.

    PARAMETER DESCRIPTION
    data

    Percorso del file di cui si vuole determinare la codifica.

    TYPE: Path

    RETURNS DESCRIPTION
    str

    La codifica rilevata. Se viene restituito 'ascii', viene automaticamente sostituito con 'latin1'.

    TYPE: str

    Notes
    • chardet fornisce una stima heuristica della codifica; non è infallibile.
    • La lettura è limitata ai primi 100.000 byte per migliorare le prestazioni.
    • 'latin1' è una scelta sicura per evitare errori su file con caratteri accentati o codifiche ambigue tipiche dei dataset amministrativi ISTAT.

    csv_from_excel(data, output_path, metadata=False)

    Converte un file Excel (.xls) in formato CSV.

    La funzione legge un file Excel in formato legacy (.xls) tramite xlrd e converte il contenuto di un foglio in un file CSV. Se metadata=True, viene convertito il foglio chiamato "Metadati"; in caso contrario, viene convertito il primo foglio disponibile escludendo "Metadati" se presente.

    La conversione mantiene l'ordine delle righe e scrive tutti i campi con csv.QUOTE_ALL per garantire compatibilità e preservare delimitatori, stringhe con spazi o caratteri speciali.

    PARAMETER DESCRIPTION
    data

    Percorso del file Excel da convertire.

    TYPE: Path

    output_path

    Percorso del file CSV prodotto.

    TYPE: Path

    metadata

    Se True converte il foglio "Metadati". Se False converte il primo foglio utile escludendo "Metadati" (default: False).

    TYPE: bool DEFAULT: False

    RETURNS DESCRIPTION
    Path

    Percorso del file CSV generato.

    TYPE: Path

    RAISES DESCRIPTION
    FileNotFoundError

    Se il file Excel indicato non esiste.
    XLRDError

    Se il file non può essere letto o il foglio richiesto non esiste.
    Exception

    Per eventuali altri errori durante la conversione o la scrittura.
    Notes
    • La conversione utilizza xlrd, pertanto il file deve essere in formato .xls (Excel legacy). I file .xlsx non sono supportati da xlrd.
    • Il CSV viene salvato in UTF-8.
    • La funzione utilizza tqdm per mostrare una barra di avanzamento.

    get_census_dictionary(census_year, region_list=[])

    Genera i link ufficiali ISTAT per dati censuari, geodati e confini amministrativi.

    La funzione costruisce dinamicamente i percorsi di download in base all’anno di censimento e alla lista delle regioni desiderate. Gestisce le differenze strutturali fra censimenti precedenti (1991–2011) e quello del 2021.

    PARAMETER DESCRIPTION
    census_year

    Anno del censimento (1991, 2001, 2011 o 2021).

    TYPE: int

    region_list

    Lista delle regioni per cui generare i link dei geodati. Se vuota → regioni 1–20.

    TYPE: List[int] DEFAULT: []

    RETURNS DESCRIPTION
    dict

    Dizionario contenente i link: - data_url per i dati del censimento - geodata_urls per le basi territoriali - admin_boundaries_url per i confini amministrativi - census_code codice primario per join e identificativi

    TYPE: dict

    RAISES DESCRIPTION
    ValueError

    Se l’anno fornito non è supportato.

    get_region(region_list=[])

    Restituisce la lista delle regioni da utilizzare per il download dei geodati.

    Se non viene fornita una lista, restituisce l’elenco completo delle 20 regioni italiane (codici 1–20). In caso contrario, restituisce la lista fornita.

    PARAMETER DESCRIPTION
    region_list

    Lista dei codici delle regioni da utilizzare. Se vuota → restituisce tutte le regioni (1–20).

    TYPE: List[int] DEFAULT: []

    RETURNS DESCRIPTION
    List[int]

    List[int]: Lista dei codici regione da processare.

    remove_files(files_path)

    Rimuove una lista di file dal filesystem.

    PARAMETER DESCRIPTION
    files_path

    Lista di oggetti Path da eliminare.

    TYPE: list

    Notes
    • Le eccezioni non vengono catturate: se un file non può essere eliminato, l’errore emerge esplicitamente (comportamento desiderabile nel workflow ETL).

    unzip_data(input_data, output_folder)

    Decomprime un file ZIP nella cartella di destinazione specificata.

    La funzione apre un archivio ZIP e ne estrae l’intero contenuto nella cartella indicata. Se la cartella di output non esiste, viene creata automaticamente. Funziona come componente interno del workflow di download dei dati ISTAT.

    PARAMETER DESCRIPTION
    input_data

    Percorso del file ZIP da decomprimere.

    TYPE: Path

    output_folder

    Cartella in cui estrarre il contenuto dell’archivio.

    TYPE: Path

    RETURNS DESCRIPTION
    Path

    Il percorso della cartella contenente i file estratti.

    TYPE: Path

    RAISES DESCRIPTION
    FileNotFoundError

    Se il file ZIP non esiste.
    BadZipFile

    Se il file fornito non è un ZIP valido.
    Exception

    Per qualsiasi errore durante la decompressione.

    census_trace(file_path, year, output_path=None)

    Estrae il tracciato dei metadati dal foglio “Metadati” di un file Excel.

    La funzione accede al foglio denominato "Metadati" di un file Excel relativo ai dati censuari, ne estrae le colonne fondamentali (nome del campo e descrizione) e costruisce un DataFrame pandas con indice basato sul nome del campo. Se viene fornito un percorso di output, il tracciato viene salvato anche in formato CSV.

    PARAMETER DESCRIPTION
    file_path

    Percorso del file Excel da cui estrarre i metadati.

    TYPE: Path

    year

    Anno di riferimento del censimento, utilizzato per generare il nome del file di output.

    TYPE: int

    output_path

    Percorso della cartella in cui salvare il CSV del tracciato. Se None, viene restituito direttamente un DataFrame.

    TYPE: Path DEFAULT: None

    RETURNS DESCRIPTION
    Path

    Path | pd.DataFrame: - Il percorso del file CSV generato, se output_path è fornito. - Un DataFrame pandas contenente il tracciato dei metadati, se output_path è None.
    RAISES DESCRIPTION
    FileNotFoundError

    Se il file Excel specificato non esiste.
    XLRDError

    Se si verifica un errore durante l’apertura o la lettura del file Excel.
    Exception

    Per eventuali errori non previsti durante il parsing o il salvataggio.

    read_xls(file_path, census_code, output_path=None)

    Legge un file Excel (.xls) e restituisce un DataFrame oppure salva i dati in CSV.

    La funzione apre un file Excel in formato .xls, seleziona automaticamente il primo foglio utile (escludendo eventuali fogli denominati "Metadati"), estrae le righe del foglio, costruisce un DataFrame pandas e imposta come indice la colonna corrispondente al codice censuario fornito.

    Se viene indicato un percorso di output, il DataFrame viene salvato in formato CSV; altrimenti, viene restituito direttamente.

    PARAMETER DESCRIPTION
    file_path

    Percorso del file Excel da leggere.

    TYPE: Path

    census_code

    Nome della colonna da utilizzare come indice del DataFrame (es. codice ISTAT del comune).

    TYPE: str

    output_path

    Percorso della cartella in cui salvare il CSV risultante. Se None, il DataFrame viene restituito senza salvare nulla.

    TYPE: Path DEFAULT: None

    RETURNS DESCRIPTION
    Union[DataFrame, Path]

    Union[pd.DataFrame, Path]: - Un DataFrame contenente i dati letti dal file Excel, se output_path è None. - Il percorso del file CSV salvato, se viene specificato output_path.
    RAISES DESCRIPTION
    FileNotFoundError

    Se il file indicato non esiste.
    XLRDError

    Se si verifica un errore durante la lettura del file Excel.
    Exception

    Per qualsiasi errore non previsto durante il parsing o il salvataggio.

    read_xlsx(file_path, output_path=None)

    Legge un file Excel (formato .xlsx) e lo converte in un DataFrame Pandas. Se specificato, salva i dati in formato CSV.

    PARAMETER DESCRIPTION
    file_path

    Il percorso del file Excel da leggere.

    TYPE: Path

    output_path

    Il percorso in cui salvare il file CSV generato. Se non viene specificato, restituisce il DataFrame.

    TYPE: (Path, opzionale) DEFAULT: None

    RETURNS DESCRIPTION
    Union[DataFrame, Path]

    Union[pd.DataFrame, Path]: Restituisce un DataFrame se output_path è None,
    Union[DataFrame, Path]

    altrimenti restituisce il percorso del file CSV salvato.
    RAISES DESCRIPTION
    FileNotFoundError

    Se il file Excel specificato non viene trovato.
    ValueError

    Se il file Excel non può essere letto correttamente.
